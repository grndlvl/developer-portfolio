# grndlvl logo usage guide

## Preferred apparel sponsor mark

Use the compact mark for jerseys, embroidery, and sponsor grids. It contains the prompt caret, grndlvl name, and cursor, with no small tagline or decorative rule.

| Placement | Recommended file |
| --- | --- |
| Dark apparel or background | `sponsor-mark-light-print.svg` |
| Light apparel or background | `sponsor-mark-black-print.svg` |
| Full-color use | `sponsor-mark-color-print.svg` |
| Full-color use with a fixed navy background | `sponsor-mark-color-on-dark-print.svg` |

Files ending in `-print.svg` have outlined lettering and are the versions to send to printers. They do not require the grndlvl font.

## Square badge

Use the badge for squarer sponsor slots — grids where neighboring logos are crests, circles, or other icon-style badges rather than wide wordmark strips. It sets the `>_` prompt glyph above the grndlvl name in a square (220×220) composition.

### Which cursor goes where

The cursor takes a different form depending on the format, and this is deliberate:

| Format | Cursor | Reasoning |
| --- | --- | --- |
| Horizontal mark and descriptor lockups | Block `▮` after the name | Reads as the name typed at a prompt |
| Square badge, favicon, and app icon | Underscore `>_` | A self-contained prompt glyph that leads a stacked composition |

The badge uses the same `>_` glyph as `glyph-bare.svg` and the site favicon, so the apparel badge and the browser tab icon match. Do not swap one cursor form for the other to make the two families look identical — they are separate formats, not variants of one another.

| Placement | Recommended file |
| --- | --- |
| Dark apparel or background | `sponsor-badge-light-print.svg` |
| Light apparel or background | `sponsor-badge-black-print.svg` |
| Full-color use | `sponsor-badge-color-print.svg` |
| Full-color use with a fixed navy background | `sponsor-badge-color-on-dark-print.svg` |

Use the horizontal `sponsor-mark-*-print.svg` files instead when the sponsor slot is a wide strip rather than a square.

## Descriptor lockups

The longer `sponsor-lockup-*.svg` files are for headers, presentations, email signatures, and larger digital placements where **APPLIED AI · SOFTWARE · WEB** remains readable. Do not use the descriptor lockup for small apparel or sponsor-row placements.

| Descriptor placement | White file | Black file |
| --- | --- | --- |
| Horizontal | `sponsor-lockup-white-print.svg` | `sponsor-lockup-black-print.svg` |
| Stacked | `sponsor-lockup-stacked-white-print.svg` | `sponsor-lockup-stacked-black-print.svg` |

## Usage

- Preserve the original proportions. Never stretch, compress, rotate, crop, or redraw the logo.
- Keep clear space around the logo equal to at least the height of the pink prompt mark.
- Keep the apparel mark at least 1.5 in (38 mm) wide for print. Confirm the final minimum with the decorator because garment and production methods vary.
- Do not add shadows, outlines, gradients, or other effects.
- Use `sponsor-mark-light-print.svg` on dark garments and `sponsor-mark-black-print.svg` on light garments.
- Confirm a physical proof before the full production run, especially for embroidery.

## Brand colors

| Color | Hex | Use |
| --- | --- | --- |
| Navy | `#0d1224` | Dark background and dark one-color mark |
| Green | `#16f2b3` | Primary wordmark |
| Pink | `#ec4899` | Prompt mark |
| Slate | `#cbd5e1` | Descriptor text |
| White | `#ffffff` | One-color mark on dark apparel |
| Black | `#000000` | One-color mark on light apparel |

Exact spot-color matching should be agreed with the printer using a physical swatch or proof.

## Production notes

- **Screen printing:** Use one of the compact one-color sponsor marks unless a full-color print has been approved.
- **Embroidery:** Use one of the compact one-color sponsor marks and ask the decorator to confirm the smallest reproducible size with a sewn proof.
- **Direct-to-garment or transfer:** The full-color print file may be used when the process supports the brand colors accurately.
- **Digital sponsor graphics:** Use `sponsor-lockup-on-dark-print.svg` when a self-contained dark background is needed.

## Descriptor

The approved descriptor is **APPLIED AI · SOFTWARE · WEB**. Do not alter its wording or order within the logo.
