# Client / project logos

Drop logo files here and I'll wire them into the **Selected Impact** section
(a monochrome "trusted by" row on the enterprise card, plus small marks on the
GroScale / NCLC / Zivtech cards).

## Format
- **Prefer SVG.** PNG at 2× (≥ 80px tall) is fine as a fallback.
- **Monochrome / white preferred** so they sit cleanly on the dark theme. If the
  files are full-color, that's OK — I'll normalize them with a CSS treatment
  (white/greyscale at reduced opacity, full color/opacity on hover).
- Transparent background.
